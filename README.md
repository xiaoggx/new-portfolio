## English

### Overview
Not just a static site, but a dynamic **Full Stack** portfolio built with **Next.js** and **PostgreSQL**. With a backend for managing the messages system, with filtering for keywords and etc.

### Features
- **Dynamic Contact Forms**: That allows message submissions with dashboard access to view, search and delete.
- **Dockerized** environment and isolated networks.
- **Responsive Design**: Mobile compatibility and Dark/Light mode.
- **Internationalization**: Seamless EN/PT language switching.

### Stack
- **Frontend**: Next.js, TypeScript, Tailwind CSS.
- **Backend**: API Routes, PostgreSQL.
- **DevOps**: Docker Compose.
